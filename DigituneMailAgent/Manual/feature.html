特徴

  • 全て Java 言語を用いて製作されていますので、JDK1.1 以上の API を持つ JavaVM のあるところでしたらどこでも実行可能 (だと思います)。実際に Win32 環境、Linux 環境での動作実績があります。
  • 「ダイアルアップ接続の個人ユーザ」をターゲットに作成しています。残念ながら電話回線の制御までは出来ませんが、ダイアルアップユーザが便利に感じるよう基本的に設計しているつもりです。
  • 設計方針を一言で言うなら「必要最小限の機能を軽量・コンパクトに」
  • 全てのテキスト系コンポーネントに K.Yasumatsu さんの作成された「KFC」を用いています。そのため高速で安定した動作となりました。
  • それなりの GUI を持ってます。またキーボードでの操作もそれなりに考慮しています。
  • MIME によるヘッダエンコーディングやマルチパートメールにある程度対応しています。uuencode や Binhex にはまだ対応していません。
  • マルチアカウントに対応、複数のメールボックスを同時にチェックできます。
  • メールのフィルタリング (自動仕分け) が可能です。先仕分け、後仕分けどちらでも OK！
  • 新規・返信・転送メールそれぞれに対してテンプレートを作ることが可能です。
  • メール保存数などには制限はありません。また、メールの数が多くなってしまった時でも軽快な動作となるよう、最大限努力しています。
  • アドレス帳機能も一応あります。
  • いわゆる UNIX mbox 形式でのインポート／エクスポートに対応しています。
