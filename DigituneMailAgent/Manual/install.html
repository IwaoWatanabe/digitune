インストール方法

DMA をインストールするには、次の手順に従ってください。

  1. まず、JDK or JRE (1.1 以降) をセットアップして、Java コマンド他が動作するようにしてください。
  2. DMA のアーカイブを適当なディレクトリで展開しておいてください。
  3. KFC 同梱版以外の場合は、KFC のアーカイブも同様に適当なディレクトリで展開しておいてください。またメール検索に正規表現を使用したい場合は、同じく GNU Regexp パッケージを展開しておいてください。

■Java2 (JDK1.2) 以降をお使いの方

その１. ちょっと上級者向け

  1. まず、KFC の jar ファイル「kfc.jar」を、利用する JavaVM の lib/ext フォルダにコピーします。
  2. メール検索に正規表現を使う場合も同様に GNU Regexp パッケージの jar ファイル「gnu-regexp-x.y.z.jar」を lib/ext フォルダにコピーしておきます。
  3. その後、DMA を展開したディレクトリで次のようにコマンドラインから入力することで起動できます。 
    % java -jar morning.jar
    また、Windows 環境では morning.jar ファイルを単にダブルクリックするだけでも起動可能です。

その２. 初心者向け

  1. まず KFC の jar ファイル「kfc.jar」と、(正規表現を利用する場合は) GNU Regexp の jar ファイル「gnu-regexp-x.y.z.jar」を、DMA のアーカイブを展開したディレク トリにコピーします。
  2. その後、DMA を展開したディレクトリで次のようにコマンドラインから入力することで起動できます。 
    % java -cp kfc.jar:gnu-regexp-x.y.z.jar:morning.jar jp.or.din.digitune.morning.MainWindow
    なお、-cp オプション後の jar ファイルの区切り記号は、UNIX 環境では「:」、Windows 環境では「;」です。

■JDK1.1 をお使いの方

KFC 同梱版以外では、まず KFC の配布アーカイブを入手してください。その後、KFC のアーカイブの中にある、「kfc.jar」ファイルを DMA を展開したディレクトリにコピーします。もし、正規表現パッケージを利用したい場合は、同様に GNU Regexp パッケージを入手後、「gnu-regexp-1.0.6.jar」を同じディレクトリにコピーしてください。

続いて、次のようにコマンドを入力することで起動できます。 

(正規表現パッケージを利用しない場合)
% jre -cp kfc.jar:morning.jar jp.or.din.digitune.morning.MainWindow

(正規表現パッケージを利用する場合)
% jre -cp kfc.jar:morning.jar:gnu-regexp-1.0.6.jar jp.or.din.digitune.morning.MainWindow

jar ファイルの区切り文字は、Windows 環境では「;」、UNIX 環境では「:」です。上記コマンドをバッチファイル等にしておくと便利です。
